Viewing a dataset as a table

The easiest way to figure out what the rows and columns in your dataset represent will often be to view the rows and columns in a table.

Required packages

This example uses the following packages:

library(tigris)
library(here)

Loading example data

Ill start by loading a set of census tracts using the tigris package.

tracts <- tracts(state = "Illinois",
                 county = "Cook")

Viewing your data

Once youve loaded the data, youll see its name on your environment tab in RStudio, along withe number of rows (observations or obs.) and the number of columns (variables)

If you click the blue icon next to the name of the data set, youll see a list of the variables, and values for the first few rows.

If you click on the name of the data set, RStudio will open a tab showing the data in a table.

You can also open that tabular view by typing View(tracts) into your console (note that the V is capitalized).
